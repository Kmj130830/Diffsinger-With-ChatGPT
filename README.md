# DiffSinger With ChatGPT

A beginner-friendly Colab project for training DiffSinger voicebanks for OpenUtau.

## Goals

- Stable on Google Colab
- Japanese HTK (.lab) support
- Automatic dataset validation
- Automatic config generation
- Resume training
- OpenUtau export
